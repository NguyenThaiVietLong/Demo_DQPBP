# Hướng dẫn chạy Demo

## Bước 1: Mở trang web

👉 **Click vào file `login.html`** trong thư mục này

Trình duyệt web sẽ tự động mở lên.

---

## Bước 2: Đăng nhập

Sử dụng một trong các tài khoản sau:

| Tên đăng nhập | Mật khẩu   |
| ------------- | ---------- |
| `admin`       | `admin123` |
| `ntnhan`      | `123456`   |
| `ppphu`       | `123456`   |

---

## Gặp lỗi?

Nếu trang web không hiển thị đúng, hãy thử:

1. **Click chuột phải** vào file `login.html`
2. Chọn **"Open with"** (Mở bằng)
3. Chọn **Google Chrome** hoặc **Microsoft Edge**

---

✅ **Xong!** Bạn có thể sử dụng demo.
